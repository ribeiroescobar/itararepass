# 🎫 Itararé Pass - Inteligência Turística e Economia Circular
### *Transformando aventura em desenvolvimento local através de tecnologia e dados.*

O **Itararé Pass** é uma Startup **TurTech/GovTech** operando via PWA (Progressive Web App). Nossa missão é resolver o déficit do "Turismo de Passagem" e combater a informalidade hoteleira, integrando atrativos naturais ao comércio urbano via gamificação georreferenciada.

---

## 🚀 1. A Tese de Negócio (Value Proposition)
Cidades com polos de ecoturismo sofrem com a desconexão econômica: o turista visita o atrativo natural e retorna no mesmo dia, sem consumir na cidade. O Itararé Pass cria uma **Flywheel de Economia Circular**:
1.  **Aventura Georreferenciada:** O turista faz check-in em pontos naturais (GPS + QR).
2.  **Conquista de Valor:** O progresso desbloqueia benefícios reais (até 25% OFF) no comércio local.
3.  **Fomento à Formalidade:** Benefícios "VIP" exigem check-in em hotéis oficializados (ISS retido no município).
4.  **Big Data Governamental:** Gestores públicos recebem insights reais sobre fluxo e ROI de marketing.

---

## 🛠️ 2. Stack Tecnológica (Enterprise Grade)
O projeto utiliza o estado da arte do ecossistema Web e AI para garantir performance em baixa conectividade:

*   **Frontend:** [Next.js 15](https://nextjs.org/) (App Router) + [React 19](https://react.dev/).
*   **Inteligência Artificial:** [Firebase Genkit](https://firebase.google.com/docs/genkit) + Google Gemini (Storytelling histórico dinâmico).
*   **Geolocalização:** [Leaflet.js](https://leafletjs.org/) com camadas de cache para funcionamento **Offline-First**.
*   **Backend:** [Firebase](https://firebase.google.com/) (Firestore, Auth, App Hosting).

---

## 🛡️ 3. Diferenciais e Moats (Barreiras de Entrada)

### Validação Híbrida Antifraude
Diferente de guias estáticos, implementamos a **Prova de Presença (Proof of Presence)**:
*   **Geofencing (GPS):** O scanner só é liberado se o turista estiver no raio de 150m do local.
*   **Token Físico (QR Code):** Exige a digitalização de uma placa física no local, impossibilitando check-ins remotos.

### AI Storytelling
Utilizamos IA para converter dados frios em experiências emocionais. Após o check-in, o Gemini gera um insight histórico único, aumentando o valor percebido da visita.
